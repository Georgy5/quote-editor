# README

Learn Hotwire with Turbo Rails tutorial https://www.hotrails.dev/turbo-rails

* Ruby version
  Ruby 3 and Rails 7

* System dependencies
  Yarn package manager
  gem install rails -v 7.0.0
  bin/setup

* Database creation
  bin/rails db:create

* Database initialization
  bin/rails db:seed

* How to run the test suite
  Run "bin/rails s"
